## 📖 프로젝트 개요

EdenForge는 완전히 자율적인 생태계와 문명이 진화하는 관찰형 시뮬레이션입니다. 플레이어는 직접 조작하지 않고, AI 엔티티들이 자체적으로 생존, 발전, 진화하는 과정을 관찰합니다.

### 🎯 핵심 특징

- **완전 자율성**: 모든 엔티티가 AI로 자율 행동
- **프로시저럴 진화**: 유전, 학습, 문화를 통한 진화
- **복합 생태계**: 동물, 식물, 파벌 간 상호작용
- **실시간 관찰**: 6개 패널로 다양한 정보 확인
- **성능 최적화**: 5,000+ 엔티티 지원

## 🎯 게임 플레이

### 📊 관찰 패널

1. **통합 패널** 📊: 재료/엔티티 정보 및 검색
2. **파벌 패널** ⚔️: 파벌 관리, 멤버, 관계
3. **유전 패널** 🧬: 유전 특성, 학습 경험
4. **생태계 패널** 🌿: 동물/식물, Pulse 관리
5. **통계 패널** 📈: 실시간 게임 통계
6. **로그 패널** 📝: 주요 사건 실시간 로그
7. **편집 패널** ✏️: 엔티티/동물/식물/재료/파벌 데이터 수정/생성
8. **설정 패널** ⚙️: 게임 저장/불러오기, 새로하기, 설정 관리
9. **파라미터 패널** 🔧: 게임 밸런스 파라미터 편집

### 🎮 조작법

- **마우스 드래그**: 패널 이동
- **패널 리사이즈**: 패널 가장자리/모서리 드래그로 크기 조정
- **탭 클릭**: 패널 열기/닫기
- **아이템 클릭**: 상세 정보 모달
- **속도 조절**: 0.1x ~ 4x
- **일시정지**: 게임 정지/재개

### 🌟 주요 기능

#### 🧬 유전 시스템
- **유전자 전달**: 부모의 특성이 자식에게 전달
- **돌연변이**: 랜덤한 유전적 변화
- **후성유전**: 환경에 따른 유전자 발현 변화

#### 🧠 학습 시스템
- **경험 학습**: 행동에 따른 스킬 향상
- **가르치기**: 엔티티 간 지식 전달
- **스킬 트리**: 8가지 스킬 체계

#### ⚔️ 파벌 시스템
- **파벌 생성**: 리더 중심의 그룹 형성
- **문화 발전**: 공격성, 협력성, 혁신성, 전통성
- **전투 시스템**: 파벌 간 전쟁, 동맹

#### 🌿 생태계 시스템
- **동물 AI**: 포식, 경쟁, 생존
- **식물 성장**: 성장, 번식, 환경 적응
- **Pulse 시스템**: 공포, 위험, 매력, 먹이 신호

#### 💭 감정 시스템
- **감정 계산**: 행복, 공포, 분노, 호기심 등
- **이모지 말풍선**: 행동, 감정, 생각, 대화 표시
- **상호작용**: 전투, 대화, 거래, 교배 시 감정 표현

## 🚀 기술 스택

### Frontend
- **React 18** + **TypeScript**
- **Vite** (빌드 도구)
- **Canvas 2D** (렌더링)
- **CSS Grid/Flexbox** (UI 레이아웃)

### 아키텍처
- **모듈화 설계**: core/ (로직) + components/ (UI)
- **타입 안전성**: TypeScript로 모든 타입 정의
- **성능 최적화**: 말풍선 제한, 렌더링 최적화
- **확장성**: 시스템별 독립적 개발

### 성능
- **현재**: 24개 엔티티 기본 테스트
- **목표**: 5,000개 엔티티, 60fps
- **최적화**: Web Worker 마이그레이션 예정

## 🛠️ 개발 환경 설정

### 필수 요구사항
- Node.js 18+
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/insukim/EdenForge.git
cd EdenForge

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# GitHub Pages 배포
npm run deploy
```

### 개발 스크립트

```bash
npm run dev          # 개발 서버 실행
npm run build        # 프로덕션 빌드
npm run preview      # 빌드 결과 미리보기
npm run deploy       # GitHub Pages 배포
```
